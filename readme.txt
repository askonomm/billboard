A billboard-style slider block for the WordPress block editor.

== Description ==

If you've ever wanted to add a little bit of flair to your site, this is the block for you. Billboard Slider is a simple slider block that allows you to add a series of images to your site in a billboard-style animation.

== Features ==

- Supports any number of images
- Supports custom image sizes
- You can change the height of the slider
- You can change the transition speed of the slider
- You can have the slider automatically transition
- You can change the interval speed of the automatic transition
- You can toggle the navigation arrows

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/billboard-slider` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
